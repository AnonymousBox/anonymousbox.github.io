About

Anonymousbox was concieved of at Hampshire College in Advanced sculpture Spring 2014. Anonymousbox is a large metal box which allows users to send messages to the next person who uses the box. While the box promps and invites a passerby to interacte with it the box is covertly mining data about the user.

Collecting not only the users interactions with the box. The box also collects pictures, distance readings, and temperature readings. The box is consuming much more that the user thinks they are inputting. At the end of the interactions all the information is bundled and sent to anonymousbox.herokuapp.com.

Why

Social media sites pose as innocuous, benign services which allow users to share and manipulate content. Behind this front is systemetic data collection. Most services like this are opt-out not opt-in. Only the ones who are aware that their data is being collected can choose to opt-out This piece means to show every step of that process.

How

The part of this box which the user sees is a ST7565 screen and an old PS/2 keyboard. Hidden in the box is an arduino duemilanove which handles the users interactions. Also inside, an old gateway lt3103u laptop gutted of its harddrive and booting xubuntu from a flash drive. The laptop collects data from the arduino and captures three images using U19-A Webcam concealed behind the front frame.

Code

  • The front end of the project was coded with arduino using the adafruit ST7565 and PS/2 keyboard library.
  • The backend was made using express on top of node, with angularjs to display the messages and search for them.
  • The laptop in the box is running a python script which takes pictures with the camera and uploads the images and data from the arduino to the website
