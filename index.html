About

Anonymousbox is a large metal box which allows users to send messages to the next person who uses the box. While the box promps and invites a passerby to interacte with it the box is covertly mining data about the user. Collecting pictures, ultrasonic range finder readings, and temperature, the box is consuming much more that the user thinks they are inputting. At the end of the interactions all the information is bundled and sent to anonymousbox.herokuapp.com.

Why?

Social media sites pose as innocuous, benign services which allow users to share and manipulate content. The intense data collection is hidden from the users of such websites. This piece means to show every step of that process.

From the simple technology (the arduino and old keyboard) used to gather data to the more complex technology used to parse and display (the laptop and node.js website)

How?

The small screen and old keyboard are powered by an arduino duemilanove. An old laptop running xubuntu on a flash drive handles sending the data gathered by the arduino and webcams to the website. All the code for this project can be found here
